# React Native - Personal project for the Udemy course

This is the client source code for the GoTonight website.

###Getting Started###

Create firebase config.js  file in src directory

```
export const config = {
  firebase: {
    apiKey: '...',
    authDomain: '...',
    databaseURL: '...',
    storageBucket: '...',
    messagingSenderId: '...'
  }
};
```

npm i
